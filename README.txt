Summary:
A Synology Video Station plugin that pulls (AV) movie info from javdb.com.

Note:
For it to work, name the movie with ID (something like SSIS-001).

Usage:
0. Put all the files into a directory called com.synology.TheJavDb
1. Zip the directory into com.synology.TheJavDb.zip
2. Go to Video Station -> Settings -> Video Info Plugin -> Add
3. Upload the zip file



简介：
用于群晖Video Station的AV信息拉取插件。信息源为javdb.com.

电影文件命名为番号（SSIS-001）

用法：
0. 将所有文件放入名为com.synology.TheJavDb的文件夹
1.压缩文件夹为com.synology.TheJavDb.zip
2.Video Station -> Settings -> Video Info Plugin -> Add
3.上传此压缩文件夹
